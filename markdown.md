## Списки

чтобы добавить ненумерованные списки, необходимо начать строку со звездочки (*). например вот так:
* Элемент 1
* Элемент 2
* Элемент 3

чтобы добавить нумерованные списки , необходимо пункты пронумеровать. например так: 

1. Первый элемент
2. Второй элемент
3. Третий элемент
